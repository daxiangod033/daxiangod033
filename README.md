## 🔥 我的硬核项目

### 🚁 **YdFly - 轻量级无人机飞控系统**
基于 **ESP32** 的飞控系统，让无人机飞得更稳、更智能！
- 自主导航 | 姿态解算 | 实时图传
- **核心技术**: FreeRTOS, PID控制, MPU6050, ESP-IDF
- 👉 [查看项目](https://github.com/daxiangod033/YdFly)

### 🚗 **STC89C52智能小车**
会循迹、会避障、还能蓝牙遥控的小车，我的第一个机器人作品！
- 循迹算法 | 超声波避障 | 蓝牙通信
- **核心技术**: 51单片机, PWM驱动, HC-05蓝牙
- 👉 [查看项目](https://github.com/daxiangod033/stc89c52car)

### ⚡ **YdgodOS - 微型操作系统**
基于 STM32 的微型 OS，给单片机装上"大脑"！
- 任务调度 | 内存管理(还没做出来)| 设备驱动
- **核心技术**: STM32, 汇编, C语言, 操作系统原理
- 👉 [查看项目](https://github.com/daxiangod033/YdgodOS)
